# T1D_Demo1

## Setup

1. `npm ci`
2. `npm run start`

## Scripts

- `npm run lint`: Run ESLint
- `npm run type-check`: TypeScript compilation
- `npm test`: Run Jest tests

## Architecture

- **TypeScript** + **Expo Router**
- **ESLint** / **Prettier** for code quality
- **Jest** / **Testing Library** for tests
